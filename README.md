# Clara CoderDev

Practical coding and automation workspace for building small, maintainable tools with AI-assisted development.

Current open-source focus:

- maintainer automation
- research collaboration tooling
- local-first workflow prototypes
- static tools that are easy to inspect, run, and adapt

Featured project:

- [Research Connection Engine](https://github.com/clara-coderdev/research-connection-engine) - dependency-free browser prototype for discovering adjacent research collaborators.
